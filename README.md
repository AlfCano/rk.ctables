# rk.ctables
rk.ctables is an RKWard plugin that brings the power of SPSS-style "Custom Tables" (CTABLES) to R. Powered by the [expss](https://gdemin.github.io/expss/) package, it allows you to build complex hierarchical tables using a simple drag-and-drop interface, bridging the gap for users migrating from SPSS to R.
